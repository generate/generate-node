# generate-node [![NPM version](https://badge.fury.io/js/generate-node.svg)](http://badge.fury.io/js/generate-node)

> Generate a node.js project.

WIP

## Install

Install globally with [npm](https://www.npmjs.com/)

```sh
$ npm i -g generate-node
```

## Usage

```js
$ generate node
```
